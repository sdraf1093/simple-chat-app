# simple-chat-app
A simple simple-chat-app implementation

## Features
- User authentication
- Real-time messaging

## Technologies
- Node.js
- Express
- Socket.io

## Usage
1. Clone the repository
2. Install dependencies (`npm install`)
3. Start the server (`npm start`)
4. Access the app at `http://localhost:3000`
